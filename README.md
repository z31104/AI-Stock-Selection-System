# AI 智慧股票選股與投資配置系統

本專題使用 Python Flask 建立一套股票選股與投資配置系統，結合技術指標、新聞趨勢分析、AI 文字分析、回測與資金配置功能，協助使用者進行股票篩選與投資決策參考。

## 功能特色

- 股票查詢
- MA5 / MA20 技術指標
- RSI / KD 指標
- K 線圖
- 技術評分系統
- Google News 新聞趨勢分析
- AI 分析文字
- 股票排行榜
- 股票比較
- 投資配置建議
- 現金保留比例
- 停損停利建議
- 回測系統
- 模擬投資
- AI 智慧選股推薦
- AI 投資報告

## 使用技術

- Python
- Flask
- Pandas
- Requests
- Matplotlib
- mplfinance
- feedparser
- HTML / CSS
- TWSE 股票資料 API

## 專案目標

透過股票技術指標、新聞趨勢與資金分配邏輯，建立一套可展示於作品集的 AI 投資輔助系統。

## 注意事項

本系統僅作為學習與專題展示用途，不構成任何投資建議。
